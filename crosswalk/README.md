# Control-to-regulation crosswalk

The **authoritative** control-×-regime crosswalk is **CL-044 (Chapter 18, "The Control-to-Regulation
Mapping")** in the book, with the per-regime detail in Chapters 19–23 (DORA, EU AI Act, FCA/PRA, PCI
DSS, UK/EU GDPR). The book deliberately states each regulatory obligation **once**, in Part VI, and
does not reproduce obligation text elsewhere — so this repository does **not** duplicate the obligation
cells.

`control-to-regulation.template.csv` gives you the catalogue's 28 controls as rows and the five regimes
as columns, ready to populate **from the book's Part VI** for your own register. Fill each cell with the
obligation reference the book maps that control to (e.g. for DORA, the pillar/article and the owning
section), keeping the book as the source of truth and your "as of" date alongside.
